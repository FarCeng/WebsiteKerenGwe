Project Saya

Project ini merupkana project pertama saya di perkuliahan. Project ini merupakan project yang sangatlah sederhana yang bertujuan untuk mengimplementasikan dasar-dasar dari bahasa C pada pembelajaran Pemerograman I. Project ini menggunakan Arduino Uno, Sensor ultrasonik HC-SR04, Layar LCD 16x2 I2C, Modul Micro SD Card, Micro SD. Project ini berfungsi untuk menyimulasikan lampu rambu lalu lintas untuk kendaraan bermotor dan lampu rambu lalu lintas untuk pejalan kaki.
